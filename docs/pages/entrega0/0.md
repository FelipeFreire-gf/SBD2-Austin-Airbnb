# Entrega 0 - Base de Dados

## Propósito de Avaliação

### Objetivo
Estabelecer a base de dados para análise de atrasos de voos em aeroportos dos Estados Unidos, implementando a camada **Bronze** (dados brutos) da arquitetura Medallion.

### Fonte de Dados

**Dataset**: Airline Delay and Cancellation Data (2013-2023)  
**Fonte Original**: Bureau of Transportation Statistics (BTS) - U.S. Government  
**Período de Cobertura**: Agosto 2013 - Agosto 2023 (10 anos de dados históricos)  
**Tamanho do Arquivo**: 28.73 MB (`Airline_Delay_Cause.csv`)  
**Licença**: U.S. Government Works (domínio público)  
**Usabilidade**: 10.0/10 no Kaggle  
**Atualização**: Anual

#### Propósito do Dataset

O dataset foi criado para oferecer insights detalhados sobre a performance operacional de companhias aéreas dos EUA durante 10 anos. Ele permite análises de:

- **Análise de Performance**: Avaliar pontualidade de diferentes carriers em aeroportos específicos e identificar áreas de melhoria
- **Identificação de Tendências**: Analisar tendências temporais de atrasos, cancelamentos e desvios para entender períodos com maiores desafios operacionais
- **Análise de Causa-Raiz**: Investigar os principais contribuidores de atrasos (companhia, clima, NAS, segurança, aeronave)
- **Benchmarking**: Comparar performance de várias companhias em diferentes aeroportos para identificar líderes da indústria
- **Modelagem Preditiva**: Usar dados históricos para desenvolver modelos preditivos de atrasos
- **Insights Estratégicos**: Contribuir para melhor compreensão dos fatores que influenciam eficiência operacional no setor aéreo americano

#### Estrutura do Dataset

O dataset possui **formato tabular** com **21 colunas** organizadas por combinações únicas de:
- Ano (year)
- Mês (month)
- Companhia aérea (carrier)
- Aeroporto (airport)

Cada linha representa métricas agregadas para essa combinação específica, incluindo contagens de voos, atrasos, cancelamentos e breakdown detalhado por causa de atraso.

### Estrutura dos Dados Brutos

O arquivo `dados_brutos.csv` (`Airline_Delay_Cause.csv`) contém os seguintes 21 campos:

#### Dimensões Temporais e Identificadores

| Campo | Descrição |
|-------|-----------|
| year | Ano do registro (2013-2023) |
| month | Mês do registro (1-12) |
| carrier | Código da companhia aérea (ex: AA, DL, UA) |
| carrier_name | Nome completo da companhia aérea |
| airport | Código IATA do aeroporto (3 letras) |
| airport_name | Nome completo do aeroporto |

#### Métricas Operacionais

| Campo | Descrição |
|-------|-----------|
| arr_flights | Número total de voos de chegada |
| arr_del15 | Número de voos atrasados ≥15 minutos |
| arr_cancelled | Número de voos cancelados |
| arr_diverted | Número de voos desviados para outro aeroporto |

#### Contagem de Atrasos por Causa

| Campo | Descrição |
|-------|-----------|
| carrier_ct | Contagem de atrasos causados pela companhia (manutenção, tripulação, etc.) |
| weather_ct | Contagem de atrasos causados por condições meteorológicas adversas |
| nas_ct | Contagem de atrasos causados pelo Sistema de Aviação Nacional (tráfego aéreo) |
| security_ct | Contagem de atrasos causados por procedimentos de segurança |
| late_aircraft_ct | Contagem de atrasos causados por aeronave anterior atrasada |

#### Tempo de Atraso por Causa (em minutos)

| Campo | Descrição |
|-------|-----------|
| arr_delay | Tempo total de atraso de chegada (minutos) |
| carrier_delay | Tempo de atraso atribuído à companhia (minutos) |
| weather_delay | Tempo de atraso atribuído a meteorologia (minutos) |
| nas_delay | Tempo de atraso atribuído ao NAS (minutos) |
| security_delay | Tempo de atraso atribuído a segurança (minutos) |
| late_aircraft_delay | Tempo de atraso atribuído a aeronave atrasada (minutos) |

### Características do Dataset

- **Granularidade**: Agregado por ano, mês, companhia e aeroporto
- **Formato**: CSV (Comma-Separated Values)
- **Codificação**: UTF-8
- **Total de Registros**: ~500.000+ linhas
- **Período**: 120 meses de dados (10 anos)
- **Cobertura Geográfica**: Todos os principais aeroportos dos EUA
- **Cobertura de Carriers**: Todas as companhias aéreas comerciais americanas
